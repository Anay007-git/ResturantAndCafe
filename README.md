# Presto Guitar Academy - Professional React Website

A modern, AI-powered guitar learning website built with React, featuring interactive components and professional design.

## Features

- **AI Guitar Tutor**: Interactive chat assistant for guitar learning
- **Chord Library**: Visual chord diagrams with professional guitar images
- **Modern Design**: Glass morphism effects and smooth animations
- **Responsive Layout**: Mobile-friendly design
- **Professional Images**: High-quality guitar photography from Unsplash

## Technologies Used

- React 18
- Framer Motion (animations)
- Lucide React (icons)
- CSS-in-JS styling
- Modern ES6+ JavaScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
